Code for the BJM package.
Based on the paper 'Li, Wenhao, Liang Li, Brad C. Astor, Wei Yang, and Tom H. Greene. (2023). Backward Joint Model for the Dynamic Prediction of Both Competing Risk and Longitudinal Outcomes. arXiv preprint arXiv:2311.00878'.
'Example_code.R' gives an example without competing risks.
'Example_code_competing_risks.R' gives an example with competing risks.
Example data is from Mayo Clinic Primary Biliary Cirrhosis Data.
